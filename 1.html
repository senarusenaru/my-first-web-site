HTML이란 무엇인가?

Hypertext Markup Lanuage(HTML) is the standard markup language for creating web pages and web applications. Web browsers receive HTML documents from a web server or from local storage and render them into multidmedia webpages.

HTML elements are the building blocks of HTML pages. With HTML constructs, images and other objects, such as interactive form, may be embedded into the rendered page.
